# 💼 Tech Jobs Analytics

Projeto de análise de vagas de tecnologia desenvolvido com Python, Pandas, Plotly e Streamlit.

O objetivo do projeto é transformar dados de vagas de emprego em insights relevantes sobre o mercado de tecnologia, utilizando técnicas de limpeza, transformação, visualização e análise de dados.

---

# 🚀 Objetivos do projeto

Este projeto foi desenvolvido com o objetivo de praticar:

- Python para análise de dados
- Manipulação de dados com Pandas
- Criação de dashboards interativos
- Estruturação de projetos reais
- ETL básico
- NLP básico para extração de tecnologias
- Regex para classificação de senioridade
- Organização modular de código

---

# 📊 Funcionalidades

O dashboard permite:

✅ Visualizar vagas de tecnologia  
✅ Filtrar vagas por localização  
✅ Filtrar vagas por senioridade  
✅ Filtrar vagas por tecnologias  
✅ Analisar as tecnologias mais requisitadas  
✅ Analisar distribuição de senioridade  
✅ Visualizar evolução temporal das vagas  
✅ Explorar insights do mercado tech
